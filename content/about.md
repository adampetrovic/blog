# About

I'm Adam Petrovic, an engineer based in Sydney, Australia.

I write about software engineering, coding, Kubernetes, homelab infrastructure, automation, and the practical lessons that come from running real systems at home.

My homelab is a GitOps-managed Kubernetes environment built around Talos Linux, Flux CD, Helm, Kustomize, SOPS, Rook-Ceph, VolSync, Envoy Gateway, and a pile of self-hosted services. It is where I test ideas, break things, fix them, and turn the useful parts into repeatable patterns.

This blog is intentionally small and plain. No tracking, no heavy frontend, no unnecessary machinery. Just notes, decisions, failures, fixes, and write-ups that might be useful to someone else building similar systems.

Expect posts about:

- Kubernetes and GitOps
- Homelab operations
- Self-hosting
- Home automation
- Developer tooling
- Reliability and observability
- Practical engineering trade-offs

If something here reads like a field note, that's probably because it is.
